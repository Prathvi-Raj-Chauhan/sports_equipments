# Sports Equipments Management System

## Overview

A complete Sports Equipment Management System built using Java, Maven, and SQL for managing inventory, issuing equipment, and maintaining records.

## Features

* Add and manage sports equipment
* Issue equipment to users
* Track returns and maintain logs
* Database-backed inventory system

## Tech Stack

* **Java (JDK 21)**
* **Maven**
* **SQL Database (MySQL)**
* **JDBC**
* **JSP**
